Configurar API Key

Para usar esta aplicación, necesitas una API Key de Google Maps:

  1. Ve a Google Cloud Console
  2. Crea un proyecto o selecciona uno existente
  3. Habilita "Maps JavaScript API"
  4. Crea una clave de API
  5. Reemplaza "YOUR_API_KEY" en el código con tu clave
